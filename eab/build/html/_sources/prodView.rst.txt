EasyBuy's Product View Page Documentation
=========================================

This documentation page shows all the details of the **product view page** of our EasyBuy e-commerce web-appication.

 * In this product view page details of an individual product will be shown.
 * The name, image, price and description of each product in the database will be shown.
 * There will be two buttons added below.
   1) Buy now.
   2) Add to cart.

The pathway of navigating to this page is,

  **Product list** -> Select **"Quick View"** buttom -> **Product view page**
